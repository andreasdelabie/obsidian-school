To create a new class folder:
- [ ] Duplicate the `/Templates/_Class` folder
- [ ] Give it a name
- [ ] In this new folder, right click the `_Class` note, go to `Folder note commands`, and click `Turn into folder note for {folder name}`
- [ ] Now, click on the folder title. This is the homepage for your new class!
- [ ] Enter your teacher's name in the properties section
